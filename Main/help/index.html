Хаос-магическая программа визуализации уничтожения различных объектов.

 

Инструкция пользователя.

 

Общие положения.

 

1. Назначение программы.

Программа предназначена для использования в Хаос-магических ритуалах в качестве средства визуализации уничтожения различных объектов тварного мира, мемов, психоконструктов, образов, иллюзий, страхов и т.д.

 

2. Принцип работы.

С теоретическим обоснованием и описанием принципа работы программы можно ознакомиться в статье Магия и компьютерные технологии.

 

3. Запускаемые модули программы:

А. Запускаемый модуль для Windows

B. Запускаемый модуль Adobe Flash

С. Модуль, предназначенный для загрузки в программу изображений, созданных пользователем.

 

4. Системные требования.

А. Компьютер с процессором не ниже Pentium III 700 или совместимый, 256 Мб RAM, 20 Мб на жестком диске.

B. Для запуска Windows-версии программы необходима операционная система не ниже Windows XP SP2,

C. Для запуска флэш-приложения любая операционная система оснащенная браузером с  Flash-плеером не ниже версии 11.2 или сам Flash-плеер.

D. Для обеспечения работы  модуля загрузки пользовательских изображений необходим Microsoft .NET Framework 2.0. Загрузить его можно с официального сайта компании Microsoft. У пользователей операционных систем Windows Vista и Windows 7 он включен в состав операционной системы. Пользователям Windows XP может потребоваться его загрузка.

 

5. Установка

Распакуйте архив с программой в отдельную папку. После распаковки программа готова к работе.

При использовании установщика, созданного в виде самораскрывающегося (SFX) архива, программа будет распакована в указанную пользователем папку (по-умолчанию C:\Chaossoft\Visualiser) а в меню Пуск будут созданы ярлыки. Пользователям Windows 7 и Vista не рекомендуется устанавливать программу в Program Files, т.к. это может воспрепятствовать работе модуля загрузки пользовательских изображений. Возможно, будет требоваться наличие у пользователя прав локального администратора.

 

6. Удаление программы.

В случае установки с помощью установщика программа может быть удалена посредством Панели управления Windows (средство «Установка и удаление программ»), или запуском программы удаления (Ярлык находится в папке ярлыков программы в меню Пуск)

Для удаления программы, если она была установлена вручную из ZIP-архива, удалите вручную папку программы и все созданные при установке ярлыки (при наличии).

 

 

 

Работа с программой

 

  1. Запустите программу, соответственно настроившись на ритуал.
  2. В главном меню программы можно включить или отключить звук кнопкой со значком громкоговорителя.
  3. Для начала работы нажмите кнопку «Старт»
  4. В следующем меню выберите способ уничтожения вашей жертвы, либо с помощью стрелок справа и слева и нажав кнопку «Выбрать», либо отдайтесь воле Хаоса, нажав кнопку «Случайно»
  5. Аналогичным способом поступите и с объектом для уничтожения
  6. Следуйте подсказкам на экране

 

Загрузка собственных изображений

 

Внимание! Оптимальное разрешение для изображения - 320 X 240 пикселей! Иное разрешение изображения может привести к его некорректному отображению.

 

Данное действие можно осуществить двумя способами – автоматически, воспользовавшись модулем загрузки изображений (AddImage.exe) или вручную.

Модуль позволяет выбрать на компьютере любое изображение формата JPG, PNG или GIF и загрузить его в программу (кнопка «Добавить изображение»), удалить ненужные, или обновить данные об используемых изображениях (см. ниже)

Для того, чтобы загрузить изображение вручную скопируйте его в папку victims, которая находится в папке с программой и пропишите имя этого изображения (с расширением) через запятую без пробела после конструкции victims= (должна присутствовать обязательно) в файле victims.txt в папке victims

В названиях файлов настоятельно не рекомендуется использовать русские буквы.

Модуль загрузки автоматически транслителирует названия файлов.

 

Возможные неисправности

 

Неисправность: На экране выбора средства уничтожения ничего не появляется или появляются не все средства уничтожения.

Причина: Отсутствуют или повреждены файлы программы

Устранение: Загрузите программу с нашего сайта по-новой

 

Неисправность: Аналогичное явление происходит с изображениями объектов для уничтожения.

Причина 1.:  Поврежден файл изображения

Устранение: Попробуйте взять другой файл или воспользоваться целой версией файла

Причина 2.:  Поврежден файл с данными об изображениях

Устранение: Попробуйте запустить модуль загрузки изображений (AddImage.exe), нажмите на кнопку «Обновить данные», утвердительно ответьте на вопрос и перезапустите программу

 

Неисправность: Нет звукового сопровождения.

Причина 1: Не включен звук

Устранение: Включите звук

Причина 2: Отсутствуют или повреждены звуковые файлы

Устранение: Загрузите программу с нашего сайта по-новой

 

Неисправность: Другая неисправность

Устранение: Сообщите разработчику программы на e-mail hige.wolf.krl@gmail.com
